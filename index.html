Portfolio

Ultimate Package Simulator

Winner of the CCC 2017 game Jam game. My role was primarily in level design, althought I did help with the programming here and there. Ultimate Package Simulator (UPS for short) is a game about delivering packages while biking around a "living, breathing city" while avoiding cars and pedestrians against the clock. Although fairly basic, the core programming and level design in this game are pretty developed, especially for a 24 hour game jam.

Gnome Invader

Another game jam game, I contributed heavily to the design and theming of this game. The gameplay and art design are pretty rudimentary, but the main ideas are solid

Cursed

A text adventure game I made for my first semester final project, Cursed is about being trapped in a haunted house, but as the player plays through the game they gradually find out that something else is going on. I did all of the programming and came up with all of the ideas for this game, and it was built in about two weeks.
